# 🎮 Tic Tac Toe (C++ Console Game)

A simple 2-player Tic Tac Toe game built using C++.  
It runs in the terminal and includes a match history system to track winners and draws.

---

## ✨ Features

- 👤 Player name input
- ❌⭕ 2-player turn-based gameplay
- 🧠 Win detection (rows, columns, diagonals)
- 🤝 Draw detection
- 📜 Match history system
- 📊 Simple menu interface

---

## 🛠️ Concepts Used

- Classes & Objects (OOP)
- Arrays
- Structs (Record types)
- Vectors (Lists)
- Constants
- Basic Input/Output
- Game loop logic

---
